Frequently Asked Questions

Honest answers about what ByBO is — and what it isn't

Is ByBO really free?

Yes. ByBO is a guide, and every tool it recommends — a code editor, your web browser, and hosting such as GitHub Pages — is free to use. There is nothing to buy and no subscription.

Do I need to know how to code?

No prior experience is needed. The guide introduces HTML, CSS and JavaScript a little at a time, so you learn the basics as you build your own site.

What do I need to get started?

A computer, a web browser, and a free code editor such as Visual Studio Code. That's it — you can begin straight away.

Who owns the website I build?

You do. You write the files and keep them on your own computer. You can host them anywhere, change them whenever you want, and you are never locked in.

Does ByBO host my site for me?

No. ByBO shows you how to publish your site yourself using free hosting such as GitHub Pages, so you stay in full control of where it lives.

Is this an official product I can buy?

No. ByBO is an educational project created for the Summer School on AI at CUNEF Universidad. It exists to teach, not to sell — there is nothing to purchase.

Still curious?

The best way to understand ByBO is to work through the guide.
