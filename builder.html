1

Generate your site in Bolt

Open Bolt, paste the prompt below, and press send. Wait for it to finish building.

Open Boltbolt.new
Your prompt
Important — keep it static

Bolt often defaults to React. The prompt above tells it not to. Your finished project should contain index.html, style.css, script.js and a resources folder — no package.json or src/ folder. If you see those, reply in Bolt: "Rebuild as plain static HTML, CSS and JS only — no React, no build step."

2

Download the files from Bolt

Get the finished project onto your computer as a normal folder.

  • In Bolt, find the Download button (top-right of the editor) and click it — it saves a .zip file.
  • Open your Downloads folder, right-click the .zip and choose Extract All (or double-click on Mac) to unzip it.
  • Open the unzipped folder. You should see index.html, style.css, script.js and a resources folder.
Check before you upload

Double-click index.html — your site should open in the browser straight away. If it does, it's a proper static site and you're ready for GitHub.

3

Create a GitHub repo and upload the folder

Make a free repository, then drag your files straight in — no terminal needed.

Suggested repository name:

  • Go to github.com/new, paste the name above, keep it Public, and click Create repository.
  • On the new repo page, click Add file → Upload files.
  • Open your unzipped folder and select everything inside it — index.html, style.css, script.js and the resources folder — then drag them onto the GitHub page.
  • Click Commit changes.
New repositorygithub.com/new GitHub DesktopDrag-and-drop app
Watch out

Drag the files that are inside the folder, not the folder itself — index.html must sit at the top level of the repository, or your site won't open.

4

Publish it for free

Turn the repo into a live website with GitHub Pages.

  • Open Settings → Pages in your repository.
  • Set Source to "Deploy from a branch", pick main and / (root), then Save.
5

Tweak it (optional)

Edit the text, images and colors any time in a free code editor.
